# Présentation du Projet SIAC-IoT

## Slide 1: Introduction

- **Nom du projet** : SIAC-IoT

- **Objectif** : Fournir une solution IoT complète pour la surveillance et l'analyse des données.

- **Composants principaux** :
  - Backend
  - Frontend
  - Infrastructure

## Slide 2: Architecture du Projet

- **Backend** : Python (Flask), gestion des données et services ML.

- **Frontend** : React, interface utilisateur moderne et réactive.

- **Infrastructure** : Docker, Grafana, Mosquitto, PostgreSQL.

## Slide 3: Fonctionnalités Clés

- Surveillance des appareils IoT.

- Analyse des données en temps réel.

- Alertes et notifications.

- Visualisation des données avec Grafana.

## Slide 4: Technologies Utilisées

- **Backend** : Python, Flask, SQLAlchemy.

- **Frontend** : React, Tailwind CSS, Vite.

- **Infrastructure** : Docker, Grafana, Mosquitto, PostgreSQL.

## Slide 5: Organisation du Code

- **Backend** : Gestion des données, services ML.

- **Frontend** : Composants réutilisables, pages dynamiques.

- **Infrastructure** : Configuration des services, dashboards Grafana.

## Slide 6: Conclusion

- SIAC-IoT offre une solution complète pour la gestion des appareils IoT.

- Utilisation de technologies modernes pour une performance optimale.

- Facilité de déploiement grâce à Docker.
